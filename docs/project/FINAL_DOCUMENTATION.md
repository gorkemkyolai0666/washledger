# WashLedger Final Dokümantasyon

**Proje**: WashLedger — Çamaşırhane Operasyon Yönetimi  
**Tarih**: 2026-06-17  
**Durum**: Geliştirme tamamlandı, deployment bekliyor

## Özet

WashLedger, bağımsız çamaşırhane operatörleri için makine envanteri, tahsilat kayıtları, onarım takibi ve bakım planlama sunan B2B SaaS platformudur.

## Teknik Stack

- Frontend: Next.js 14, TypeScript, Tailwind CSS, shadcn/ui
- Backend: NestJS, Prisma ORM, PostgreSQL
- Deployment: Railway (backend), Vercel (frontend), GitHub Actions CI/CD
